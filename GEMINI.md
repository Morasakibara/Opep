# OPEP - Plateforme de Gestion de Transport Interurbain

## Stack Technique
- **Backend:** NestJS (TypeScript)
- **Web:** Next.js 14 (App Router)
- **Mobile:** React Native (en remplacement de Flutter)
- **Database:** PostgreSQL (TypeORM)
- **Cache:** Redis (BullMQ)

## Commandes Utiles
- `docker-compose up -d` : Lancer l'infrastructure (DB, Redis)
- `npm install` : Installer les dépendances du monorepo
- `npm run dev:api` : Lancer le backend
- `npm run dev:web` : Lancer le portail web
- `npm run dev:mobile` : Lancer l'app mobile
